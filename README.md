# portifolio
Projeto de um portfólio em HTML5, CSS3 e JavaScript

É boa pratica de designers, arquitetos entre outros profissionais manter um portfólio com suas atividades arquivadas. Mais recentemente isso passou a ser adotado por programadores, principalmente de front-end. O foco desde portfolio é documentar os meus projetos pessoais gerais, mas principalmente relacionadas a automação e programação. Além de ser uma apresentação de mim mesmo.

Todo esse site foi codificado por mim, em HTML5, CSS3 e JavaScript. A IDE foi o Visual Studio Code e versionamento feito em Git-GitHub. Esta foi minha primeira página e teve como segundo objetivo acrescentar meu aprendizado em front-end usando essas ferramentas como porta de entrada para códigos modernos. Isso também acrescentou meu conhecimento de programação, sendo que minha maior experiência se concentra em linguagens de Controladores Lógicos Programáveis para automação industrial.
